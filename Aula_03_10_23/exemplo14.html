Exemplo 14 CSS - width e height

As propriedades width e height são usadas para definir respectivamente a largura e a altura do componente html. Geralmente para definir esses valores usamos as unidades de medida relativa. Por exemplo:

Width

Neste exemplo estamos usando uma largura de 50% da largura da página e uma altura de 400px

Height

Neste exemplo estamos usando uma altura de 40% da altura da página e uma largura de 600px
