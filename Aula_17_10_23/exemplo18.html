O que é o grid layout ?

O grid layout é bidmensional, permitindo que o desenvolvedor possa separar o layout da pagina em linhas e colunas, diferentemente do flex box que é unidimencional.

Flexgrid ou grid layout

O flexbox e o grid layout podem ser usados em conjunto. Você pode aplicar o flexbox a elementos dentro de um layout grid. Aproveitando o melhor de dois mundos

A escolha entre flexbox e grid layout não se trata de um substituir outro, mas sim de entender as necessidades específicas do layout e escolher a técina que melhor atende a essas necessidades
