Seletores de ID

Os seletores de ID são aqueles que utilizam os identificadore únicos das tags HTML, em tese cada elemento html só pode ter 1 único ID

A sintaxe de um seletor de ID é a seguinte:

#seletor Id{propriedade:valor;}

Neste Exemplo, cada div apresentada possui 1 ID diferente. Vamos aplicar estilos diferentes para cade ID.
