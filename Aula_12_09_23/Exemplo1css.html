Seletores

Seletores de Tipo

Um seletor de tipo é aquele que seleciona um tipo de elemento HTML para estilização

Para criá-lo, devemos criar uma regra onde este elemento HTML vai ser representado pelo seletor.

Neste exemplo, utilizamos CSS para estilizar os elementos da web semântica

Exemplo de um seletor de tipo

h1{propriedade:valor;}, neste Exemplo temos o seletor h1 representando um elemento HTML de cabeçalho h1, e ele é delimitado com chaves com propriedades e valores.
