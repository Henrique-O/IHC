Cores contextuais do Bootstrap

O Bootstrap utiliza um sistema de cores contextuais para facilitar a marcação e a estilização de elementos com base em seu significado ou propósito. Essas classes de cores são usadas para dar significado visual aos elementos HTML. Geralmente utilizamos essas cores em botões, alertas, barras de navegação e outros elementos quye são utilizados em interfaces de usuários.
