CSS - cor padrão RGB e RGBA

As cores utilizadas em CSS podem variar bastantes. São muitos os padrões de cores que podem ser aplicados. Esses padrões podem ser RGB (Red Green Blue), o hsl (Hue, Saturationa, Lightness), dentre outros. aplicados a estes padrões podemos ainda aplicar um canal novo (a - que corresponde ao nível de transparencia da cor) - rgba ou hlsa

Nesta div estamos aplicando o padrão de cor RGB.

Já nesse parágrafo estamos aplicando o padrão RGBA, ou seja, com um grau de transparencia. Essa transparência é representada pela letra a e vai de 0 a 1, quanto mais próximo do 1 menor o nível de transparência

CSS - cor padrão HSL e HSLA

O padrão hsl, representa a cor em termo de matriz (tonalidade) saturação e luminosidade. No caso desse padrão os valores são dados da seguinte maneira:

H vai de 0 a 360 e cada valor corresponde a uma cor especifica. S será representado em percentual, ou seja, de 0 a 100%

Já nesse parágrafo estamos aplicando o padrão RGBA, ou seja, com um grau de transparencia. Essa transparência é representada pela letra a e vai de 0 a 1, quanto mais próximo do 1 menor o nível de transparência
