Exemplo 5 CSS - Vinculaçao Externas

A vinculação externa de arquivos CSS é a recomendada quando temos várias paginas HTML. Isso ocorre pois quando usamos este tipo de vinculação temos maior facilidade de manuteir os estilos das várias páginas

Na vinculação externa o arquivo CSS fica externo ao documneto HTML, separando em outra pasta ou até mesmo em máquina. Neste exemplo estamos usando um arquivo externo para estilizar esta página.
